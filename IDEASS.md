## 🎯 Проект №1 — Умный агрегатор вакансий + AI-помощник по откликам

**Статус:** идея (09.06.2026). Реализация — месяцы 3–5 по дорожной карте.

**Что это:**
Программа, которая собирает вакансии, фильтрует их под меня и помогает откликаться качественно — а не веером.

**НЕ спрей-бот** (массовый автоотклик = риск бана + низкое качество, бьёт по моей сильной стороне — я умею писать под конкретного заказчика). А **умный агрегатор + фильтр + помощник**:

1. **Сбор** — тяну вакансии через официальный **API hh.ru** (по-белому, не скрапинг). Приходит JSON = списки словарей.
2. **Фильтр / ранжирование** — функции + условия (Python junior, РФ, удалёнка, вилка по зарплате).
3. **AI-помощник** — LLM генерирует **черновик сопроводительного письма** под конкретную вакансию.
4. **Контроль за мной** — я просматриваю и отправляю сам. Качество высокое, в рамках правил площадок.

**Почему этот проект — мой:**
- Доказывает мою позицию — **product-engineer, который встраивает AI**.
- Использует весь целевой стек: HTTP-API + обработка данных + LLM-интеграция.
- Сам работает на мою цель — помогает мне искать работу.

**Какие навыки нужны (и где беру):**
- Работа с HTTP-API → месяц 3
- Обработка данных (списки, словари, функции, условия, try/except) → уже учу ✅
- LLM-интеграция (сопроводительные письма) → месяц 5

**Сопутствующее:** заодно обновить резюме под РФ-рынок.
