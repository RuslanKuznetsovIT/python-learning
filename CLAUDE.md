## Правила урока

Структура каждого урока:
1. Краткий обзор синтаксиса, который встретится в уроке
2. Объяснение темы — коротко, своими словами, с аналогиями из жизни
3. Пример кода — готовый файл с комментариями
4. Задание — Руслан пишет сам, без помощи Claude
5. Проверка — Руслан запускает код и показывает результат
6. Обратная связь — конкретные плюсы и минусы от Claude
7. Коммит + пуш

Коммиты делаем каждые 20 минут во время урока.
