**Что меняется в плане:**
1. **Месяцы 1–2 не трогаем** — Python-основы и функции нужны в любом случае.
2. **Месяцы 3–4 — переразверстать**: вместо «структуры данных + ООП + файлы» сразу FastAPI + базовый CRUD + первый сервис, дёргающий любой внешний HTTP-API (универсальный навык, не привязка к конкретному LLM).

4. **Появляется правило артефактов**: каждый месяц = один рабочий проект, который можно показать. Портфолио строится с нуля.
5. **Сжатие срока**: 12–14 мес → 8–10 мес. реалистично при текущей регулярности.

**По английскому (уточнено 20.05.2026):**
- ❌ Разговорный английский для интервью — **отменён** (РФ-рынок не требует)
